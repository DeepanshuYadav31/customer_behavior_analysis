# 📊 Customer Behavior Data Analytics Project

## 📌 Overview

This project focuses on analyzing customer shopping behavior using Python, SQL, PostgreSQL, and Power BI.
The complete analytics workflow includes data loading, cleaning, exploratory data analysis (EDA), SQL querying, dashboard creation, and report generation.

The goal of this project is to extract meaningful business insights from raw customer data and present them through interactive visualizations and reports.

---

## 📂 Dataset

* Dataset contains customer shopping and purchasing information.
* Includes features such as:

  * Customer demographics
  * Purchase history
  * Product categories
  * Spending patterns
  * Shopping behavior

The dataset was processed and analyzed to identify trends, customer preferences, and business insights.

## ⚙️ Project Workflow

* Imported dataset into Python using Pandas.

### 3. Exploratory Data Analysis (EDA)

Performed detailed analysis using:

### 5. Dashboard Development

Built an interactive Power BI dashboard including:

### 6. Reporting & Presentation

## 🚀 How to Run the Project

### 1. Clone the Repository

```bash
git clone <repository-link>
```

### 2. Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn psycopg2
```

### 3. Run Jupyter Notebook

```bash
jupyter notebook
```

### 4. Execute SQL Queries

* Open PostgreSQL Server
* Import the cleaned dataset
* Run the provided SQL queries

### 5. Open Power BI Dashboard

* Open the `.pbix` file in Power BI Desktop
